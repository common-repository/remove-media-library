This plugin removes images, videos, audios and documents from WordPress Media Library fast and easy.

== Description ==

This plugin removes Media Library files from the WordPress Media library in an easy and massive way.
You can delete thousands of files in a few minutes.

== Installation ==

1. Upload 'remove-media-library' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions == 

= Is this plugin free? =
Yes, it is

= Can I remove a large amount of images and other media files? =
Yes, this plugin is capable of remove a large number of files without time-out errors even in cheap shared hosting with very low memory resources.

= Are the files permanently deleted or will they go to the trash? =
If the WP Trash is enabled, the files will go to it. If it is not enabled they will be permanently deleted.

= Can I enable WordPress Media Trash? =
Yes, this plugin can do it for you.
